## 关于 ShopNC 溢汇商城 版
溢汇商城 是基于 ShopNC 2014 版本制作而成的商城，功能上 好商城 v3-b11 (2015 年 7 月) 版本更完善。  

## ShopNC 的特色
1. 基于 MVC 规范易于二次开发
2. 国际化，提供多语言支持
3. B2B2C 全模式电商平台，可用于 B2B，B2C，C2C 场景
4. 提供手机版
